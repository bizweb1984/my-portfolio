Project Information

  • Name: GPS-Denied Drone Localization System with visual sensors.

Project Overview

Developed a real-time visual-inertial navigation system (VINS) for drones operating in GPS-denied environments, integrating camera sensors, IMU, and SLAM algorithms to provide high-accuracy localization and mapping.

Key Features

Visual-Inertial Odometry (VIO) Integration

Fused data from stereo/monocular cameras and IMU to track drone motion with <10 cm positional error in indoor and outdoor GPS-denied areas.

Real-Time SLAM Pipeline

Implemented a SLAM system achieving 30+ FPS on embedded platforms (Jetson Nano / NanoPC T6) for continuous mapping and navigation.

Loop-Closure Detection

Integrated visual loop-closure to reduce drift by 35% over extended flight trajectories (~500 m).

Feature-Based Tracking

Used ORB / FAST features to reliably track 20,000+ keypoints per second, maintaining stability under low-light and high-motion conditions.

Trajectory Accuracy Improvement

Achieved 40% improvement in localization accuracy compared to baseline visual-only odometry by combining IMU data and camera features.

Embedded Deployment

Optimized SLAM and VIO algorithms to run efficiently on Jetson Nano and NanoPC T6, reducing CPU usage by 30% while maintaining real-time performance.

Optical Flow Motion Estimation

Implemented optical flow module that improved flight stability by 25% in GPS-denied environments.

3D Mapping & Visualization

Generated high-resolution 3D maps in real-time for autonomous navigation and mission planning.

Multi-Sensor Calibration

Calibrated camera-IMU extrinsics with sub-millimeter precision, ensuring accurate sensor fusion.

Robustness Across Environments

Successfully tested in indoor warehouses, urban areas, and forests, maintaining consistent localization performance across all conditions.
